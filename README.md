## Usage

Access via jsDelivr CDN:
```https://cdn.jsdelivr.net/gh/peterbono/moto-data@main/data/[brand].json```

Example:
```https://cdn.jsdelivr.net/gh/peterbono/moto-data@main/data/bmw.json```
